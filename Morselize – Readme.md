Morselize is a fictional meal subscription web application designed to simplify home cooking through chef-inspired meals, flexible plans, and a clean, responsive user experience. This project was created as part of a web development course and designed with scalability and real-world best practices in mind.

Project Overview

Morselize allows users to:

Explore featured meals and culinary specialties
Learn about the service through media-rich content
View the project proposal and business concept
Contact the service using an accessible, validated form
The application emphasizes responsive design, accessibility, and clean front-end architecture.
Technologies Used
HTML5 – Semantic markup for structure and accessibility
CSS3 – Custom styling and responsive layout
Bootstrap 5 – Grid system, components, and form controls
JavaScript (ES6) – User interaction and form validation

Key Features

Responsive layout optimized for desktop, tablet, and mobile
Accessible navigation and form controls
Bootstrap-powered UI components
Media page utilizing the <picture> element for responsive images
Clean, modular codebase suitable for future expansion
Accessibility Highlights
Semantic HTML (header, nav, main, section, footer)
Proper label-input associations in forms
Keyboard navigation with visible focus states
Descriptive alt text for meaningful images
Minimal, intentional use of ARIA
